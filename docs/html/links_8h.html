Description détaillée

Gestion des liens (hard links et symbolic links) dans le système de fichiers.

Ce fichier d'en-tête définit les fonctions pour créer et gérer les liens entre fichiers, incluant à la fois les liens physiques (hard links) et les liens symboliques (soft links).
