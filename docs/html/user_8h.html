Description détaillée

Gestion des utilisateurs du système de fichiers.

Ce fichier d'en-tête définit la structure des utilisateurs et les fonctions pour gérer leur cycle de vie (création, suppression, modification). Il inclut également les différents types d'utilisateurs et leurs permissions.
