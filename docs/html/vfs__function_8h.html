Description détaillée

Fonctions du système de fichiers virtuel (VFS)

Ce fichier d'en-tête contient les déclarations des fonctions principales du système de fichiers virtuel, incluant la gestion des répertoires, les opérations de lecture/écriture, et les fonctions de chemin.
